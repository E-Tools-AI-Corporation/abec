# Abec™ Licensing FAQ

> This Markdown is provided for readability. The **authoritative** version is
> [`legal/Abec-Licensing-FAQ.pdf`](legal/Abec-Licensing-FAQ.pdf). See the
> [EULA](EULA.md) and [Abe™ Runtime Redistribution License](ABE-RUNTIME-REDISTRIBUTION-LICENSE.md)
> for the controlling terms.

### Do I retain ownership of my code?
Yes. You retain ownership of all software and other works you create using Abec™.

### Can I sell software built with Abec™?
Yes. Commercial sale, licensing, and distribution of software built with Abec™ are
permitted.

### Can I deploy Abe™ applications without runtime royalties?
Yes. The Abe™ Runtime is free to deploy. E-Tools does not charge runtime royalties,
deployment fees, per-user fees, or per-installation fees.

### Can I redistribute the Abe™ Runtime royalty-free with my application?
Yes. Applications built with Abec™ may be distributed together with the Abe™ Runtime in
accordance with the Abe™ Runtime Redistribution License. This includes applications that
incorporate the runtime through static linking or dynamic linking.

### Do I need a paid license to use Abec™ commercially?
Yes. Commercial use of the Abec™ compiler requires a valid commercial license unless
otherwise authorized by E-Tools.
